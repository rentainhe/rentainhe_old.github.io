Python中的Pickle模块主要的用途在于，方便一些复杂的数据类型的保存与加载，如果仅仅保存到txt文件中，只能保存为str类型的数据，这在反序列化的时候是非常不方便的，这里对pickle的功能进行简单的学习

Pickle API Reference

  • pickle.dump(obj, file): 将python的数据转换并保存到pickle格式的文件内
  • pickle.dumps(obj): 将python数据转换为pickle格式的bytes字串
  • pickle.load(file): 从pickle格式的文件中读取数据并转换为python类型
  • pickle.loads(bytes_object): 从pickle格式的bytes字串转换为python类型

可以看到pickle.dump和pickle.load对应, pickle.dumps和pickle.loads对应
