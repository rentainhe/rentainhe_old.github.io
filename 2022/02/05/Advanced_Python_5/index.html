Lambda Expression

Lambda表达式的简单用法: lambda arguments: expression, arguments表示传入的参数, expression是这个函数中进行的计算

Lambda结合Map Function

map的用法: map(func, seq), 对这个sequence中的每个元素都进行func(x)操作

Lambda结合Filter Function

filter的用法: filter(func, seq), 其中func必须return True或者False, filter会保留sequence所有为True的结果

Lambda结合Reduce Function

reduce的用法： reduce(func, seq), reduce会对所有的元素根据func作一个累积
