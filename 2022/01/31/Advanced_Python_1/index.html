List

简单用法

  1. x = list.pop(), 移出最后一个元素并返回
  2. list.sort(), 改变原来list内的值的顺序，如果不希望改变原list内的顺序，而是得到一个新的list的话，可以使用x = sorted(list)
  3. list.remove(name), 移除某个指定的元素
  4. 快速创建包含5个0的list: list = [0] * 5
  5. 快速组合两个list: list1 + list2
  6. 数组切片: list[1:5] 返回index[1]到index[4]的数字
  7. 按一定的step访问list: list = [::2], 数组反转: list = list[::-1]

需要注意的一些细节

列表直接赋值(a=b)的话共享内存，对b的操作也会影响到a

几种解决办法:

  • 调用内置的list函数: list_b = list(list_a)
  • 使用copy方法: list_b = list_a.copy()
  • 使用切片: list_b = list_a[:]

Tuple

Tuple特性: ordered, immutable, allows duplicate elements

简单用法

  • 内置tuple接口创建tuple: mytuple = tuple(["Max", 28, "Boston"])
  • 检查是否包含某个元素: if "Max" in mytuple
  • 检查某个元素的数量, tuple.count(name):
  • 检查某个元素的index
  • 同样也可以像List一样，进行索引切片: my_tuple[::-1]

小细节

  • 创建tuple的时候只有一个元素，不加逗号的话，是str类型:
  • 使用tuple给变量赋值

使用tuple的好处: More Efficient

  • 占用的内存比List小
  • 创建tuple的速度比list要快很多

Dict特性: Key-Value pairs, Unordered, Mutable

简单用法

  • 两种方式创建dict

细节注意

unordered, mutable, no duplicates

基本用法

  • 创建set
