变量

  • bool: True or False (1byte)
  • char: 字符类型, 需要用单引号表示, 例如’a’, ‘b’ (1byte)
  • int: 2^-31 ~ 2^31 4byte
  • float: 1.23, 2.5, 1.235e2, 表示单精度浮点数，精度较低，有效位数为6-7位 (4byte)
  • double: 和float类型一样，精度更高，有效位数15-16位 (8byte)
  • long long: 表示更长的整数型，-2^63 ~ 2^63 -1 (8byte)
  • long double: 比double精度还高，有效位数18-19位 (16byte)

字节

  • 1Byte = 8bit
  • 1char = 1Byte = 8bit，有一个字节，可以表示8位01的组合，表示2^8共256位数

计算

+ - * / %

  • 5 / 2 = 2
  • 5.0 / 2.0 = 2.5

取模的结果取决于前面的数的正负，这点和数学定义很不一样

  • 5 % 2 = 1
  • -5 % 2 = -1

变量的强制转化

  • int to float and double: 可以直接转换
  • float double to int: 下取整
  • float to double: 没问题
  • double to float: 损失精度
  • char和int也可以互相转换: 需要有个ASCII码表，char本质上是存的整数
  • a*b这个计算，会自动把里面精度小的类型转换为两数之间精度更高的那个，比如float * double -> double

编译相关

  • Accepted: 编译正确
  • Wrong Answer: 错误
  • Time Limit Error: 超时
  • Memory Limit Error: 超内存
  • Segmentation Fault: 一般可能是数组越界
