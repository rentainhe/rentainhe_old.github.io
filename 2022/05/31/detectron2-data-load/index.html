get_detection_dataset_dicts

在build_detection_train_loader的第一步是调用get_detection_dataset_dicts, 这个方法需要传入几个参数:

  • name: a dataset name or a list of dataset names
  • filter_empty: 过滤掉没有instance / annotations的images
