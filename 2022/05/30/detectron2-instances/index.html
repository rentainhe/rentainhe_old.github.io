Instances

主要的功能在这个class的docstring里有描述:

  • represents a list of instances in an image
  • stores the attributes of instances (e.g., boxes, masks, labels, scores) as “fields”.
  • All fields must have the same __len__ which is the number of instances
  • non field attributes of this classes are considered private

Basic Usage

一些基本的Instances的类方法:

  • has: 判断是否包含某个field
  • remove: 移除某个field
  • get: 得到某个field
  • get_fields: 返回整个Instances中的所有fields, 是一个dict类型
  • 举几个例子说明一下基本方法的使用

Advanced Usage

  • to: 对每个field都调用to方法, 类比于torch的to方法, 可以to("cuda")
  • __getitem__: 定义了可以通过索引操作得到Instances中的元素
  • cat: 将一个Instances列表给cat到一起
